# Distributed-system
分布式压力阵列系统
## 主要功能
一个主机通过CAN总线连接八个从机，主机通过w5500将数据上传到上位机，实现阵列压力的分布式采集。
## 硬件部分
包含三个文件  
主机板子  
从机板子  
w5500的测试板
